#### Summary

Climb Credit is a fin-tech company based in New York that provides easy
credit evaluation for educational loans in the USA.

I joined them as a freelancer to help build the e-commerce interface
enabling students to pay installments against their loans.

#### Responsibilities

- converting figma designs to react components
- making improvements to the design system
- code reviews
- rest api integration
- responsive design

#### Challenges

The greatest challenging in this project was handling credit-card data securely .

In the US, credit card companies have very strict (legally binding) protocols for handling data for which a specific license is required which is also very costly.

Therefore, our solution was to use a third party company that provided iframes to handle the data securely.

This proved to be challenging on the front-end as I was boxed in by their API and styling. At the same time, I was required to bring their iframes to be as close to Climb Credit’s design system.

Unmounted / Re-mounting of React components also proved to be a challenging aspect of this, as the **iframes** would disappear when the component unmounted. Since the documentation was also poor, I had to tinker with their JavaScript api and create a workaround that integrated with react seamlessly.

My takeaway from this was it’s important to go through all the possibilities and most often you’ll find away to solve any problem.
